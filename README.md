# Audio Loop Player

語学学習に特化したオーディオループプレーヤー。波形表示とドラッグによるA–Bループ選択が特徴。

## 機能

| 機能 | 説明 |
|---|---|
| ファイルブラウザ | ディレクトリをナビゲートしながら音声ファイルを一覧表示 |
| 波形表示 | ファイルの波形をリアルタイム再生位置インジケーター付きで表示 |
| A–Bループ | 波形をドラッグして繰り返し再生するリージョンを設定。リリース時にA点へ自動シーク |
| シーク | 波形をクリックして任意の位置にジャンプ。ループリージョン外へのクリックはループを自動解除 |
| 速度調整 | 0.25× 〜 2.0× で再生速度を変更（ピッチも変化） |
| 音量調整 | 0 〜 2.0×（増幅対応） |
| スペースキー | 再生 / 一時停止 |

### 対応フォーマット

MP3 / WAV / FLAC / OGG / M4A / AAC / Opus / WMA

## 使い方

```
# 開発ビルド
cargo run

# リリースビルド（推奨）
cargo run --release
```

### 基本操作

| 操作 | 動作 |
|---|---|
| ファイルクリック | 読み込んで再生 |
| スペースキー | 再生 / 一時停止 |
| 波形クリック | クリックした位置にシーク |
| 波形ドラッグ | A–Bループリージョンを選択（リリース時にA点へ移動） |
| 波形右クリック | ループリージョンをクリア |
| Loop ボタン | ループのオン/オフ切替 |

## 依存クレート

| クレート | 用途 |
|---|---|
| [eframe / egui](https://github.com/emilk/egui) | GUIフレームワーク |

## アーキテクチャ

```
src/
├── main.rs      エントリーポイント
├── audio.rs     オーディオエンジン（デコード・再生・アトミック共有状態）
├── theme.rs     カラーテーマ定義
└── app.rs       UI（ファイルブラウザ・波形ウィジェット・コントロール）
```

UIスレッドとcpalオーディオコールバックスレッドの間の共有状態は、ホットパスにMutexを使わず**アトミック変数**のみで管理しています。デコード済みサンプルバッファは `Arc<Vec<f32>>` でラップし、コールバッククロージャに参照カウントのみで共有しています。

## 既知の制限

- 速度変更はピッチも変化します（タイムストレッチ非対応）
- 現在のファイルを変更した場合、再読み込みは手動で行う必要があります
